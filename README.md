# MERN Blog App

A simple Blog Application built using the MERN stack.

## Live Demo
https://spontaneous-croquembouche-f7c557.netlify.app/

## Features

- View all blogs
- Add a new blog
- Edit existing blogs
- Delete blogs
- MongoDB database integration
- REST API using Express.js

## Tech Stack

### Frontend
- React
- React Router
- Axios
- CSS Modules

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
